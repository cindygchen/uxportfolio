devCircle

UX Designer & Front-End Developer

devCircle is a project that I created while taking my full stack coding bootcamp at UC San Diego. DevCircle is a web application that helps bootcamp students practice the technologies they’ve learned in school as well as keep up with new technologies by collaborating with their peers on group projects.

I had the opportunity to utilize my UX skills with this project as well as practice my newly developed coding skills

  • Created user task flows, IA, sketches and styleguides
  • Created all visual elements including logos, graphics, style guides, etc
  • Developed front-end components and pages
